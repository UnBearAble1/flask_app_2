You can operate, manipulate, and hack into computer systems. If you don’t have physical access to a computer system’s user interface, you must use a hacking kit to access and manipulate the system. 

Computers are set up to give one or more authorized users “root access,” allowing them to access any information or function of the computer as a standard action, with no need for a Computers check. Firewalls can block off specific sections of a computer and grant different users root access to those sections.

Untrained Actions:

  • Navigate Unsecured System: You can use Computers to access an unfamiliar, alien, or obscure unsecured computer system’s most basic functions. Standard use of standard computers typically don't require checks. 

Trained Actions: 

  • Craft Computer Components: You can custom build computers, software or Daemons using the crafting rules. 
  • Create Electronic Forgery: You can use Computers to forge official electronic documents (like official electronic mail, secured document files, or deepfakes) or credentials (like passwords, key cards or biometric data).  
  • Detect Electronic Forgery: When observing an electronic forgery, you make a roll versus your computer skill to detect it as such. 
  • Infiltrate Secured System: You can use Computers to hack a computer system to which you don’t already have access. This action is composed of multiple levels. 
  • Avoid System Detection: If a system is on alert, you can avoid detection using the Computers skill. 
  • Bypass Countermeasures: Whether it's a static trap or an active countermeasure, you can use the Computers skill to avoid the consequences.
