Quick Copy


Cantrip, Conjuration
Traditions:
arcane, divine, primal, occult
Two Actions; material, somatic, verbal
Range; Touch Target: One object of no greater than 1 Bulk Duration:
10 minutes

You create a short-lived copy of an item you are touching. If the object copied is non-magical, then copy functions as the original did. If the object is magical, then the copy is a non-usable replica. When the spell ends, the copy fades immediately from existence.
