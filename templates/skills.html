Skills


These effects are around which skills you are specialized in. If you are looking for bonuses to activities normally associated with skills, check out Enhanced Activity Use instead.

Additional Performance or Lore Specialty

[Mod: 4]

You become specialized in a new performance, art or lore.

Additional Skill Specialty

[Mod 4*]



You become specialized in a new skill.

*Each additional time you select this effect, add Mod +4 points to its cost, cumulatively. So the first skill you gain would be Mod 4, the second Mod 8, the third Mod 12, then Mod 16 and so on.

Dynamic Skill


[Base: 12, Mod +8]
Activation: Duration

You become specialized in a skill, performance, art or lore you weren't specialized in. The duration ends if you use the ability again.

Modification:

  • This is the same skill each time (Base -8, Mod -8)

Share Skill Specialization


[Base: 8]
Activation: Duration
[Mental], Default Duration: 10 minutes
Default Range: Touch

You can grant a creature specialization with a skill that you are specialized in.

Modification:

  • Performance, art, or lore only: Base -4

Unspecialized Improvisation

[Base: 4, Mod +2]

Instead of having downgraded results with unspecialized skills, you are -5 to the roll.
