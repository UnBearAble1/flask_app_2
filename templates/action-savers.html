Action Savers


These effects save you an action or actions. As the three action economy is a substantial part of how you will tactically behave in combat, reducing the number of actions an activity takes is a substantial boon.

Not all actions are equal. When calculating the action saver costs, unless otherwise specified, apply the following modifiers:

  • Category A: Frequently Used Actions: Like Stride, Step, Strike, Reload, Take Cover or activating powers
  • Category B: Less Frequently Used Actions: Like Stand, Interact for a specific purpose like drawing a weapon or consuming a potion
  • Category C: Rarely Used Actions: Like "Point Out"
  • Skill Based Activities (Use the categories listed in Enhanced Activity Use)
