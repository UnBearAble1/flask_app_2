TRAINED

Divination
Traditions occult
Two Actions ; somatic, verbal
Range touch; Targets 1 object

You place a hand on an object to learn a piece of information about an emotional event that occurred involving the object within the past month, determined by the GM. If you cast object reading on the same item multiple times, you can either concentrate on a single event to gain additional pieces of information about that event, or gain a piece of information about another emotional event in the applicable time frame.

Heightened (Expert) You can learn about an event that occurred within the last year.
Heightened (Master) You can learn about an event that occurred within the last decade.
Heightened (Legendary)  You can learn about an event that occurred within the entirety of the object’s history.
